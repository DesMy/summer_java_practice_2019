Ford-Fulkerson Algorithm Visualizer

Author: Igor Sergeev + Nguyen Quang Huy + Nguyen Thi Thu Duyen

About program

This program is created in the practice session for 2nd year students. The purpose is to get use to the programming language Java, to create a program which can visualize the calculation process of algorithm Ford-Fulkerson.

About the algorithm:

The idea behind the algorithm is as follows: as long as there is a path from the source (start node) to the sink (end node), with available capacity on all edges in the path, we send flow along one of the paths. Then we find another path, and so on.
